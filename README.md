# Usage
 Encrypt plain text (utf-8) into emojitext
 ```javascript
 var emojiText = encrypt(text);
 ```

 Decrypt emojifyed-text into utf-8 plaintext
 ```javascript
 var plainText = decrypt(emoji);
 ```
